Tầm quan trọng của việc đặt tên rõ nghĩa (Naming Convention)

Cách chúng ta đặt tên cho class và id như dau-trang, tieu-de-chinh hay menu-chinh được gọi là cách đặt tên theo vai trò nội dung. Điều này giúp loại bỏ hoàn toàn những cái tên vô nghĩa như box1, aaa, abc.

Khi dự án lớn lên, việc tuân thủ quy tắc viết chữ thường, ngăn cách bằng dấu gạch nối giúp mã nguồn CSS và JavaScript dễ đọc, dễ bảo trì, và giúp đồng đội trong nhóm hiểu ngay đoạn code đó đang điều khiển khu vực nào của trang web.
